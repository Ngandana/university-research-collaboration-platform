# Sprint Planning

## Sprint Goal

The goal of this sprint is to deliver a Minimum Viable Product (MVP) that enables users to log in, create research projects, join projects, and upload documents.

This sprint focuses on core system functionality required for basic research collaboration.

---

## Selected User Stories for Sprint

* US-001 (Login)
* US-002 (Create Project)
* US-003 (Join Project)
* US-004 (Upload Document)
* US-008 (Manage Users)

## Sprint Contribution to MVP

This sprint delivers the foundational features required for system operation. Without authentication, project creation, and document upload, the platform cannot function. Therefore, completing this sprint establishes the core functionality upon which future features such as messaging and reporting will be built.
